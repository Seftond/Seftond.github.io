Projects

  • King Me- Henry IX Music

    Demo Github
    React | CSS | Node | Express | PostgreSQL | Google Maps API | Vonage API | Formik | Yup | Nodemailer

    A website created for a Utah musician to share information with their fans. The site contains a page for users to interact with events, a shop, contact page, and a subscribe page. Using the Google Maps API, Vonage API and Nodemailer, users can send messages to the musicians email, see the location for the event, RSVP for the events and ask to be sent a text message the day before an event happens. All events and user information are stored in a postgreSQL database.

  • The Degenerate Gambler

    Demo Github
    HTML5 | CSS | Node | Express | PostgreSQL | Bcrypt | Odds-API

    This web application is designed to bring awareness to gambling habits by allowing users to track their sports betting record and practice sports betting without having to risk any money at all. The project is centered around data supplied by the Odds-API to list live sports betting odds. It includes a page where wagers can be placed and a profile page where the record is displayed. All of the user information is stored in a postgreSQL database and accessed using login credentials.
